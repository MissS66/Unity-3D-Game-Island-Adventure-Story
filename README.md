# Unity-3D-Game-Island-Adventure-Story
1. 项目简介 本项目基于 Unity 引擎，打造一座可自由探索的 3D 虚拟海岛。玩家既能选择自由点击场景探索或驾车在风景丰富的场景中随意漫游、欣赏景观，也能在探索途中主动触发趣味小游戏，自由选择是否参与。项目以 “探索 + 娱乐” 为核心，将沉浸式漫游与轻量互动游戏融为一体，带来轻松、完整、富有乐趣的虚拟体验。。 2. 核心目标 玩家可通过 WASD / 方向键自由移动，配合鼠标右键转换视角，空格上升，C下降，在海岛上无限制漫游探索 走近特定游戏区域时，自动弹出进入游戏确认面板，支持自由选择 确认后平滑切换到小游戏场景，完成闯关、收集或互动挑战 游戏结束或主动退出后，返回主漫游场景继续探索 全程配备清晰 UI 提示、背景音乐与交互音效，提升沉浸感与操作友好度
